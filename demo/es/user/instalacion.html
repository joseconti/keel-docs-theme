Instalación

Instala Basalt en tu tienda y conéctalo a tu almacén en menos de diez minutos.

En esta página

  • Instalar Basalt
  • Verificar la conexión

01Requisitos

Antes de empezar comprueba que tu entorno cumple lo siguiente:

  • PHP 8.1 o superior
  • WooCommerce 8.0 o superior
  • Un token de API de tu panel de Basalt

02Instalar Basalt

  1. Descarga el plugin

    Descarga basalt.zip desde tu panel, sección Descargas.

  2. Sube e instala

    En WordPress ve a Plugins → Añadir nuevo → Subir y selecciona el archivo.

  3. Activa

    Pulsa Activar. Se abrirá el asistente. Guarda con Ctrl + S.

Qué deberías ver

El asistente muestra el paso «Conectar almacén» con un campo para el token.

Consejo

Pega el token con Ctrl + V; Basalt valida el formato al instante.

03Verificar la conexión

Vuelve al panel de Basalt: el estado debe cambiar a «Conectado» en menos de un minuto.

Nota

El primer sondeo puede tardar hasta 60 s. Refresca si sigue en «Pendiente».
