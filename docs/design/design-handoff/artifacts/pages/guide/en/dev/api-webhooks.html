Webhooks

Receive events, transform their payload and verify authenticity.

01Subscribe to events

POST/v2/webhooks

Body

application/json
{
  "event": "stock.updated",
  "url": "https://tienda.example/webhooks/basalt",
  "secret": "whsec_..."
}

02Filter basalt_webhook_payload

Modify the payload before delivery. It runs in the plugin.

ParameterTipo/TypeDescription
$payloadarrayThe event payload.
$eventstringEvent name.
functions.php
<?php
add_filter('basalt_webhook_payload', function($payload, $event) {
  $payload['store'] = get_bloginfo('name');
  return $payload;
}, 10, 2);

03Verify the signature

Danger

Reject any request whose HMAC signature does not match. Never trust the body unverified.

webhook.php
<?php
$expected = hash_hmac('sha256', $body, $secret);
if (! hash_equals($expected, $signature)) {
  http_response_code(401);
  exit;
}
