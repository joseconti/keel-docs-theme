Inventory

Read and update the stock of each SKU in real time.

On this page

01List stock

Since v2.0

GET/v2/stock/{sku}

Parameters

ParameterTipo/TypeRequiredDefaultDescription
skustringyes—Item identifier.
warehousestringnoallFilter by warehouse.
freshbooleannofalseBypass the 30 s cache.

Example

Request
# Query stock for a SKU
curl -X GET \
  "https://api.basalt.dev/v2/stock/SKU-4821" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Accept: application/json"
<?php
$basalt = new Basalt\Client($token);
$stock = $basalt->stock('SKU-4821');
echo $stock->available;
const res = await fetch('https://api.basalt.dev/v2/stock/SKU-4821', {
  headers: { Authorization: 'Bearer ' + token }
});
const stock = await res.json();

Response

200 · application/json
{
  "sku": "SKU-4821",
  "available": 42,
  "updated_at": "2026-07-16T09:20:00Z"
}
