Installation

Install Basalt on your store and connect it to your warehouse in under ten minutes.

On this page

01Requirements

Before you start, make sure your environment meets the following:

  • PHP 8.1 or newer
  • WooCommerce 8.0 or newer
  • An API token from your Basalt dashboard

02Install Basalt

  1. Download the plugin

    Download basalt.zip from your dashboard, under Downloads.

  2. Upload and install

    In WordPress go to Plugins → Add new → Upload and pick the file.

  3. Activate

    Click Activate. The wizard opens. Save with Ctrl + S.

What you should see

The wizard shows the “Connect warehouse” step with a token field.

Tip

Paste the token with Ctrl + V; Basalt validates the format instantly.

03Verify the connection

Return to the Basalt dashboard: the status should switch to “Connected” within a minute.

Note

The first poll can take up to 60 s. Refresh if it stays on “Pending”.
