Configuration

Tune how and when Basalt syncs your inventory.

On this page
  • General settings
  • Schedule syncing

01General settings

basalt.sync_mode
Where it lives
Settings → Basalt → General
Valid values
auto · manual · scheduled
Default
auto
Effect
Controls when stock changes are pushed.
ModeDescription
autoSyncs whenever a change is detected.
manualSyncs only when you ask.
scheduledSyncs on a schedule.

02Schedule syncing

With scheduled mode, set an interval between 5 and 1440 minutes.

Caution

Intervals under 15 min on large stores may exhaust your API quota.

Advanced settings

These rarely need changing:

  • basalt.batch_size — items per batch (10–500).
  • basalt.retry — retries on error (0–5).
