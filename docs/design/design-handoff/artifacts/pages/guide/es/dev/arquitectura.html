01Componentes

Basalt se compone de tres piezas que se comunican por HTTPS.

  • Plugin — captura cambios y expone la interfaz.
  • API — la superficie REST en api.basalt.dev.
  • Worker — procesa colas y reintentos.

02Flujo de datos

Para contribuir, clona el repositorio y arranca el entorno de desarrollo:

Terminal
git clone https://github.com/basalt/basalt.git
cd basalt && composer install
npm run dev

03Seguridad

Peligro

Los tokens se guardan cifrados. Nunca los registres en logs ni los envíes a terceros.
