Inventario

Consulta y actualiza las existencias de cada SKU en tiempo real.

En esta página
  • Listar existencias

01Listar existencias

Desde v2.0

GET/v2/stock/{sku}

Parámetros

ParámetroTipo/TypeRequeridoDefectoDescripción
skustringsí—Identificador del artículo.
warehousestringnoallFiltra por almacén.
freshbooleannofalseIgnora la caché de 30 s.

Ejemplo

Petición
# Consultar existencias de un SKU
curl -X GET \
  "https://api.basalt.dev/v2/stock/SKU-4821" \
  -H "Authorization: Bearer $TOKEN" \
  -H "Accept: application/json"
<?php
$basalt = new Basalt\Client($token);
$stock = $basalt->stock('SKU-4821');
echo $stock->available;
const res = await fetch('https://api.basalt.dev/v2/stock/SKU-4821', {
  headers: { Authorization: 'Bearer ' + token }
});
const stock = await res.json();

Respuesta

200 · application/json
{
  "sku": "SKU-4821",
  "available": 42,
  "updated_at": "2026-07-16T09:20:00Z"
}

Campo/FieldTipo/TypeDescripción
