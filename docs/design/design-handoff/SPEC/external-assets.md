# external-assets.md

**none.**

No se ha identificado ningún asset que Design no pueda producir. Todo el material visual del tema es tipografía autoalojada (woff2 OFL), iconos SVG dibujados a mano y estructura CSS. No hay fotografía, ni ilustración compleja, ni renders 3D, ni nada que exija una herramienta generativa externa.

Los placeholders de figura/captura (`figure.figure__frame`) son intencionadamente rayas CSS + rótulo monospace: marcan dónde el proyecto consumidor colocará sus propias capturas reales. No son un asset a producir por Design: son un hueco del sistema que keel/el autor del proyecto rellena con su material.
